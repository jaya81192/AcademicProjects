Logistic Regression:

To run logistic regression: 

python LR.py <learning rate>

eg. 
python LR.py 0.01

The code file and the train and test files are inside the folder LR. The feature 12 of train and test data is split into binary format with 6 different columns.

Naive Bayes:

To run Naive Bayes:

python NB.py

The code file and the train and test files are inside the folder NB. 
The file 'featureDesc' contains the feature description with possible values of each feature in each line. The line number corresponds to the feature number, while reading the file in the code.
