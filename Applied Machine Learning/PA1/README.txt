Decision Tree


To run the code, follow the following syntax:
python decisionTree.py <train file name> <depth> <test file name>

eg. 

python decisionTree.py zoo-train.csv 8 zoo-test.csv

Note: Provided the train and test files are in the same folder as the code. 

To run the code on our dataset:
python decisionTree.py 1000-train-shuffled-disc.csv 2 500-test-shuffled-disc.csv
